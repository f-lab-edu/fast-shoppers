# fast-shoppers

## 프로젝트 컨셉

선착순 한정판매에 특화된 쇼핑몰로, 각종 한정판 아이템을 구매할 수 있는 쇼핑몰입니다. <br/>
한정판 특성상 아이템 하나당 한 사람만 구매 가능합니다.
